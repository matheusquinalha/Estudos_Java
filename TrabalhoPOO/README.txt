Esta é o trabalho que irá subtituir a prova da aula de Programação Oriente a Objeto do segundo bimestro do segundo ano, do curso de Sistemas de Informação.

Nele, estamos desenvolvendo um programa gerenciador de estoque, onde o usuário poderá criar diversos armazens, com diversos produtos cada, e realizar a manipulação desses dados.

A ideia é fazer a leitura e geração de um arquivo.txt a parte, para um melhor gerenciamento.
